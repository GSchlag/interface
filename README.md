# fiskaltrust-demo
Hier wird Demo-Code zur Anbindung des fiskaltrust.Service angeboten

## fiskaltrust.Service 
Der aktuelle Dummy-Service ist unter /tools/fiskaltrust.service.local/ zu finden. Die Datei fiskaltrust.service.local.exe muss mit Administratorberechtigung gestartet werden.
Optional kann als erster Aufrufparameter die Art der Signaturerstellungseinheit gewählt werden, als zweiter Aufrufparameter kann der Service-URL angegeben werden.
Per Default liegt der Serive-URL für 
- SOAP bei http://localhost:1201/fiskaltrust/POS, für 
- REST-xml bei http://localhost:1201/fiskaltrust/POS/xml und für 
- REST-json bei http://localhost:1201/fiskalturst/POS/json
